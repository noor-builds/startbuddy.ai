# 🚀 StartBuddy.ai

> Your AI Co-Founder for Building Startups

StartBuddy.ai helps founders transform ideas into real businesses by providing AI-powered guidance across validation, product development, marketing, fundraising, and growth.

---

## 📌 Overview

Building a startup is difficult. Most founders struggle with:

* Finding the right idea
* Validating market demand
* Planning product development
* Creating go-to-market strategies
* Acquiring early users
* Raising funding

StartBuddy.ai acts as an AI startup advisor that helps founders make better decisions and execute faster.

## 🚀 Getting Started

### Prerequisites

* Flutter SDK
* Node.js
* Supabase Project
* OpenAI API Key

### Installation

#### Clone the Repository

```bash
git clone https://github.com/yourusername/startbuddy-ai.git
cd startbuddy-ai
```

#### Install Backend Dependencies

```bash
cd backend
npm install
```

#### Install Frontend Dependencies

```bash
cd frontend
flutter pub get
```

#### Configure Environment Variables

Create a `.env` file:

```env
OPENAI_API_KEY=your_api_key
SUPABASE_URL=your_supabase_url
SUPABASE_ANON_KEY=your_supabase_key
```

#### Run Backend

```bash
npm run dev
```

#### Run Flutter App

```bash
flutter run
```
